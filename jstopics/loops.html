Loops

We use loops when we want to repeat some code or activity for some certain amount of times, or under certain conditions.

While Loops

Does a thing while the condition is true

While (condition){ some code; }

Stops when the code is done continue - > stops the current loop, and starts the next loop break - > stops the loop Issues: If you forget to increment, then the win condition to stop loop creates an infinite loop
