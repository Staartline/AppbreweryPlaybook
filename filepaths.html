Filepaths

These are locations for files and folders on one's computer. These are used to direct a user to a specific file on the computer.

Absolute

This is a filepath that is relative to the root of the computer.

Relative

This is a filepath that is relative to the folder that the current file is residing inside. These are shorter, and are valid as you move some folders around.

current directory is the folder in which the project file is located inside.

../ -> means to go up a level/folder

./ -> stay within the current directory

Webpages

These are the different files that have information. They go into the same folder. We must refer to these files in different files to be able to navigate them.
