CSS Animation

It is possible to animate with CSS.

  1. Inside of your styles.css file, set up a @keyframes animationname{}
  2. Inside of the @keyframes animationname{}, set up actions that happen during the action. @keyframes animationname{0%{action} 100%{action}}
  3. Choose a flass that you would like the animation inside. For example: .tetospinclass{}
  4. Inside of .tetospinclass{animation:animationname}
  5. Additionally, you can specify the duration of the animation duration and iterations inside of the .tetospinclass. -> animation-iteration-count:count; animation-duration:seconds;
