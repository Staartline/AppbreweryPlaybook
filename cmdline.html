GUI Shells and Command Line Interfaces

Bash shell is bourne again shell. This is for unix system.

Used for greater control. easy and fast to make a folder for example.

It doesnt hide things like config files.

Commands

  • ls -> lists items in a folder
  • cd -> change directory. We can navigate by using cd foldername. tab autofils foldernames.
  • can be tedius to nav to directories, so you can go the whole way.
  • Arrow up and down to use prev commands
  • cd .. takes you to prev directory
  • mkdir
  • alt click changes cursor position
  • ctrl a sends beginning, ctrl e sends to end.
  • ctrl u clears a line

    • Making Directories

  • mkdir foldername
  • rmdir
  • touch filename -> creates/opens a file
  • start filename -> opens the file
  • start -a application filename -> Opens the file in an application (MAC)
  • pwd - > shows current path
  • rm * -> remove all files using wildcard.
