Making The CSS Challenge Website

  1. Initial Setup: Create a folder called CSSSite to store the files inside.
  2. Inside CSSSite, add a file called index.html (working HTML File)
  3. index.html -> Create the HTML Boilerplate
  4. Inside CSSSite, add a folder called css.
  5. Inside CSS -> Create a styles.css file
  6. Link the styles.css file and index.html files together.
    1. Inside of the head of the index.html file, add a link to the file by using < link rel="stylesheet" type="text/css" href="css/styles.css">
