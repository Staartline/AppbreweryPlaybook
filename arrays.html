Arrays

We can use an array to store items in a list.

Features

  • Collection of related items stored in the same container
array[1] will return the item at index 1. declare -> var arrayname =[item, item, item] arrayname.includes(someitem) can be used to find out if an item is in the array. returns true or false. push adds a new item. arrayname.push(value). pushes the value to the end of the array. Pop pops item off the end of an array.

Intermediate Arrays

  • array.pop(value) -> removes the value at the end of the array
  • array.push(value) - > adds the value at the end of the array
