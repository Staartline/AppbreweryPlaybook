If else. Conditionals. if (condition){} -> does this code else{} -> does some othr thing. If the first statement is not true, then the second code runs. You can even have an elseif. We use this to control a flow of code. equality ===

Comparitors

  • === -> Equal to (checks datatypes)
  • !== -> Not equal to
  • > Greater than
  • < less than
  • >=
  • <=
  • == equal to (does not check datatype)

We can even fine tune this and combine them.

&& is and checks both || is or checks for either ! not
